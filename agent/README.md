# Agent Evolution Assets

This directory contains the cold-path governance files for the workflow.
These files are not part of routine PostgreSQL task execution.

Use them when you are:

- changing the workflow itself,
- debugging repeated workflow failures,
- deciding whether a new rule, skill, or template is justified.

Contents:

- `EVOLUTION_RULES.md`: criteria for growing or simplifying the workflow

Do not treat this directory as required reading for normal PostgreSQL coding tasks.
